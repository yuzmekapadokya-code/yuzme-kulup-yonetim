1. Koleksiyonlar ve Dokümanlar Oluştur

Firestore Güvenlik Kuralları

Firebase Console'da Firestore → Rules kısmına aşağıdaki kodu yapıştırın:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if request.auth != null; } match /users/{userId} { allow read, write: if request.auth.uid == userId; } } }

Firebase Verilerini Sıfırla
