# Admin Panel Analysis

Bu dokuman web admin panelinin mobil tasarima aktarilan ana akislarini ozetler.

## Dashboard
- Toplam sube, antrenor, ogrenci ve tahsil edilen gelir ozeti.
- Son kayitlar listesi.
- Yaklasan veya geciken taksit bildirimleri.

## Organizasyon Yonetimi
- Kulup profili: kulup adi ve logo.
- Sekreter olusturma ve silme.
- Sube CRUD.
- Antrenor CRUD, coklu sube atamasi, yorum ortalamasi.

## Programlama ve Operasyon
- Coklu antrenorlu ders saati tanimi.
- Ders tipi, baslangic tarihi, gun secimi, kapasite, ders sayisi.
- Sube bazli antrenor musaitlik sablonu.
- Antrenor musaitlik ozeti.
- Ogrenci ders bitis takvimi.
- Saat bazli fiyatlandirma.

## Finans ve Icerik
- Ek gelir ve gider kayitlari.
- Filtreli finans ozetleri.
- Duyuru gonderimi.
- Etkinlik olusturma.
- Indirim kodlari.
- Admin kapsami barajlari.
- Market urunlerini goruntuleme.

## Mobil Karsilik
- Admin ana sayfasi: dashboard + hizli moduller.
- Organizasyon ekrani: kulup, sekreter, sube, antrenor.
- Program ekrani: saatler, fiyatlar, musaitlik, bitis takvimi.
- Is ekrani: finans, duyuru, etkinlik, indirim, baraj, market.
