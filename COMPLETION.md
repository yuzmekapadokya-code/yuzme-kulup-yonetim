## 📞 KULLANIM ÖRNEKLERİ

### Yönetici: Yeni Antrenman Saat Eklemek
1. Admin paneline gir
2. "Ders Saatleri"ne git
3. "+ Yeni Saat"i tıkla
4. Şube, saat ve antrenörü seç
5. Kapasite belirle
6. "Kaydet"i tıkla

### Sekreter: Öğrenci Kaydetmek
1. Sekreter paneline gir
2. "Öğrenci Kayıt" formu doldur
3. Şube ve saat seç
4. Taksit sayısını belirle
5. "Kayıt Oluştur"u tıkla
6. Veli e-mail ve şifresi otomatik oluşturulur

### Antrenör: Kronometer Kullanmak
1. Antrenör paneline gir
2. "Antrenmanlar"a git
3. "Başlat"ı tıkla
4. Kronometer açılır
5. "Başla"yı tıkla
6. Mola yapabilir, sonraki egzersiz geçebilir

### Veli: Durumu Kontrol Etmek
1. Veli e-mail ve şifresiyle gir
2. "Özet"te öğrenci bilgileri
3. "Ödeme Durumu"nda taksitleri
4. "Devam Durumu"nda derslerini görebilir
